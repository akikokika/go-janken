# go-janken

a training program for golang.

## Usage

    $ <font color=#006699>cd</font> $GOPATH  
    $ go get golang.org/x/tools/cmd/stringer  
    $ go get github.com/akikokika/go-janken  
    $ <font color=#006699>cd</font> src/github.com/akikokika/go-janken  
    $ go generate  
    $ go build  
    $ ./janken
